# Responsive Business Website Template

A modern and fully responsive business website template built using **HTML, CSS**.
Designed for agencies, startups, and portfolio projects.

---



## Features

* Fully Responsive Layout
* Single Page Website
* Bootstrap Grid System
* AOS Scroll Animations
* Stats / Counter Section
* Case Studies / Projects
* Contact Section
* Gradient CTA Banner
* Newsletter Footer
* Clean UI Design

---

## Tech Stack

* HTML5
* CSS3

## Installation & Run Locally

Clone the repository:

```bash
git clone https://github.com/your-username/business-website.git
```

Go to project folder:

```bash
cd business-website
```

Open in browser:

```bash
index.html
```

OR run with Live Server (VS Code).

---

## Deployment

You can deploy easily on:

* GitHub Pages
* Netlify
* Vercel

### GitHub Pages Steps

1. Go to **Repo → Settings**
2. Click **Pages**
3. Select branch → `main`
4. Save → Get live link

---

## Customization

Edit content in:

```
index.html → Text & Sections
css/style.css → Colors & Layout
images/ → Replace images/logo
```

---

## Animations Usage

Example AOS animation:

```html
<div data-aos="fade-up">
```

Initialize AOS:

```javascript
AOS.init({
  duration: 1000,
  once: true
});
```
